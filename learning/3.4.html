Doubles

Double Data Type

div>

A double is used to represent a number with even more precision that a float. It often has a maximum precision of 14 decimal places so is useful for when you need to work with fractional parts of numbers. When initialising a double variable we have to put an 'd' after the double number to show it is a double.

Here is how you could initialise a double data type:


double a = 17.4d; //initialising a double point number, notice the d after the 17.4

Here is a sample code of how doubles can be used in Java. Copy and paste this into your IDE and play around with the numbers and perhaps initialise and add more floats and see what you can do with them.

Imagine you have to work with numbers where they have a high number of decimal places. The "double" data type will allow you to do this and in the example below we are simply going to add two numbers together and print the output.


public class Floats {
    public static void main(String[] args) {

        double number1 = 3.2152353d;
        double number2 = 17.35756d;
        double number3;

        System.out.println("The number is: " + number3);
    }
}

This is what your terminal should output:

The sum is 20.5727953

Make sure you try change the numbers around and see how the output differs.
