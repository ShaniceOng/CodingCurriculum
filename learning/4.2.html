Methods!

What is a method?

Java method is a group of statements that are grouped together, thus forming an operation – it can be considered analogous to what is called a ‘function’ in C. When we call the System.out.println method, the system, in the background, carries out many statements in the background to print data on the console terminal.



Each method is defined by a set of properties i.e. it can have a return type or no return type , it can take parameters or no parameters , invoke a method in the parent class by either overriding or overloading another method.

Syntax of declaring a method

Below is an example of how a method is to be defined. 

    modifier returnType MethodName (Parameters) 
    {
    // method body
    }

The method definition consists of two components: the header , containing the modifier , the return type , name of Method, and the parameters. The second part consists of the method body , the set of statements which constitutes the operation of the method.

Here is an example to explain the syntax of the method: 

  public static void method(int a, int b) 
{
  // body
}

In this case, the ‘public static’ is the modifier, ‘void’ is the return type, ‘method’ is the name of the method, a and b are formal parameters , and int a , int b are the formal parameters with their respective types (int).

Method: It defines the access type of the method, and helps when using object-oriented tools like inheritance, overloading, overriding, polymorphism, and interfaces.

Return Type: It specifies the return type (i.e. int, String, bool ), or in some situations, no return value (void).

Method Name: Method name identifies a method , however, there might more than one methods having the same name in the case of overriding.

Parameter List: The list of parameters , the types, the order, and number of parameters. These are optional however, as the method may have zero parameters. Parameters can be thought of as inputs, or arguments.

Method body: The method body is the set of statements and syntax which defines the function of the method. It determines what operation the method will carry out.

Example 2: Below is method that takes an array and sorts it in (increasing) ascending order using the Insertion Sort Algorithm. 

   public static void InsertionSort( int [ ] array)
    {
         int j;                     // the number of items sorted so far
         int key;                // the item to be inserted
         int i;  

         for (j = 1; j < array.length; j++)    // Start with 1 (not 0)
        {
               key = array[ j ];
               for(i = j - 1; (i >= 0) && (array [ i ] < key); i--)   // Smaller values are moving up
              {
                     array[ i+1 ] = array[ i ];
              }
             array[ i+1 ] = key;    // Put the key in its proper location
         }
    }

Example 3: Below is a method that takes an integer n and returns the factorial of the integers. There is another way to compute this factorial which uses recursion, which will be explained in recursion . 

  public static int factorial(int n) 
    {
        int fact = 1; // this  will be the result
        for (int i = 1; i <= n; i++) {
            fact *= i;
        }
        return fact;
    }

Example 4: Below is a method that computes the GCD (Greatest Common Divisor) of two numbers using the Euclidean Algorithm. There, again, is a recursive method of defining this method which will be explained in recursion . 

  public static int euclid(int a, int b) 
    {
        while (b != 0) {
            int temp = b;
            b = a % b;
            a = temp;
        }
        return a;
    }
