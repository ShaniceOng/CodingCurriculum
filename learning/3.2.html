Integers

Integer Data Type

The integer data type will store any integer value. It has 4 different types of values - long, int, short and byte.

Here is how you could initialise an integer data type:


long a = 15000; //initialising a long integer
int b = 1100; //initialising an int
short c = 33; //initialising a short integer
byte d = 15; //initialising a byte

This is one way you would be using integers in a java program later on. Try copy and pasting this code into your IDE and playing around with the numbers. It is important to notice you can only perform operations such as addition, multiplication etc. on integers of the same type. For example if you try to add a long integer to a short integer, you get an errror.

This is an example of how you might want to use integers in a program. We have not covered how to make a program yet but by copy and pasting the code into your IDE and changing the numbers you can start to get familiar with the concept of integers.

Let's try and show you how you might use integers in Java to solve a real life problem. Most of you must know the equation which relates "force, mass and acceleration".

"Force = Mass x Acceleration"

The following program will find the acceleration given two values for mass and acceleration which will stored as integers.


public class Integers {
    public static void main(String[] args) {

        int mass = 5;
        int acceleration = 20;
        int force = mass * acceleration;

        System.out.println("The force is: " + force + " N");
    }
}

This is what your terminal should output:

The force is: 100 N

Make sure you try change the numbers around and see how the output differs.
