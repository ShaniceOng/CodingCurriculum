Branching

Branching is used for executing a statement(s) or the other depending on a condition(s).

if
switch
 
Syntax:
if (condition)
    Statement1;
else if (condition)
    Statement2;
    :
else
    StatementN
    
Syntax:
switch(Variable)
{
    case : ;
    case : ;
    :
    case : ;
    default: ;
}
 
Allows
 - conditions with all relational operators
 - conditions with all logical operators
 - conditions of all data types
 
Allows
 - conditions with equal to/not equal to
 - conditions catering to or operator
 - conditions with char and int variables only (Java 7+ switch on Strings to used as well)
Note: Conditions with <, >, <=, >= will not be possible on switchpre>
