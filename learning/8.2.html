Search Algorithms

A Search Algorithm, like the name suggests is, an algorith for locating a particular item among a collection of items.


Linear Search

It is a logic to search for a value from an array by checking the elements one after the other starting from the very first element of the array.


Example: To search for a particular employee code from an array containing codes of all the employees in a company.

Solution:

public class Linear {
  Scanner s = new Scanner(System.in);
  
  // Method to enter the content of an array
  public void Register(long E[])
  {
    for(int i = 0; i < E.length; i++)
    {
      System.out.print("[" + i + "] Eno?");
      E[i] = s.nextLong();
    }
  }
  
  // Method to search for a particular eno (employee number)
  public void Search(long E[], long Eno)
  {
    int Found = 0;
    for(int i = 0; i < E.length && Found == 0; i++)
      if(E[i] == Eno)
        Found = 1;
    if(Found == 1)
      System.out.println(Eno + " is Found");
    else 
      System.out.println(Eno + " is not Found");
  }
  
  public static void main(String[] args) {
    // TODO Auto-generated method stub
    Linear obj = new Linear();
    long Eno[] = new long[6];
    long Enos;
    System.out.println("Enter all the Emp. Nos:");
    obj.Register(Eno);
    System.out.print("Eno to be searched: ");
    Enos = obj.s.nextLong();
    obj.Search(Eno, Enos);
  }
}

Note: In the above example, Search( ) method is to find whether a particular Eno is present in the array or not and so as soon as it is found, value of Found variable is changed to 1, and the loop terminates.
Look at the condition in the loop i < E.length && Found == 0the loop terminates either when the value of i becomes E.length or Found becomes 1.

The selection sort algorithm keeps track of the smallest element from the ith element onwards in the array. Once iterated through the entire list of elements, it compares the smallest element with the ith element and switches their position in case the ith element is not the smallest element.


Ascending Order using Selection Sort Descending Order using Selection Sort 
void selectionA(int A[])
{
  for(int I = 0; I < A.length - 1; I++)
  {
    int Small = I;
    for(int J = I + 1; J < A.length; J++)
      if(A[Small] > A[J])
        Small = J;
    if(Small != I)
    {
      int T = A[Small];
      A[Small] = A[I];
      A[I] = T;
    }
  }
}

The insertion sort divides the array into 2 portions: a sorted portion and an unsorted portion. At each step in the algorithm, a number is moved from the unsorted portion to the sorted portion till eventually the entire list is sorted. In the beginning, we assume that the first element is the start and end of the sorted portion and the rest of the list is part of the unsorted portion. In each iteration of the loop, the first element in the unsorted portion is moved to the sorted portion by finding its appropriate location in it and shifting/moving all the others elements accordingly.

Ascending Order using Insertion Sort Descending Order using Insertion Sort 
void insertionA(int A[])
{
  for(int I = 1; I < A.length; I++)
  {
    int Temp = A[I], J = I - 1;
    while(J >= 0 && Temp < A[J])
    {
      A[J + 1] = A[J];
      J--;
    }
    A[J + 1] = Temp;
  }
}
 
void insertionD(int A[])

    {
      A[J + 1] = A[J];
      J--;
    }
    A[J + 1] = Temp;
  }
}

Binary Search

It is an algorithm which starts by checking the value to be found with the middle element of the sorted array.

- If the target value and the middle element are equal, then the search finishes.
- If the target value is less than the middle element, then the search process is repeated on the left half of the array (or the right half if the array is sorted in descending order).
- If the target value is greater than the middle element, then the search process is repeated on the right half of the array (or the left half if the array is sorted in descending order).

This process continues, eliminating half of the elements on each iteration, and comparing the target value to the middle element of the remaining elements - until the target value is either found, or until the entire array has been searched.

Prerequisite: The array content should be sorted (Ascending/Descending).

boolean binarySearch(int A[], int Data)
{
  int LB = 0, UB = A.length - 1, MID;
  boolean Found = false;  

  while(LB <= UB && !Found)
  {
    MID = (LB + UB) / 2;
    if(A[MID] > Data)
      UB = MID - 1;
    else if (A[MID] < Data)
      LB = MID + 1;
    else 
      Found = true;
  }
  return Found;
}

Merging

Like the name suggests, it is a logic used to merge two or more sorted arrays to produce a third sorted array with slight changes depending on whether the two array are sorted in an ascending order, descending order or one of each.

Java Function to MERGE two ascending order array Input Array A Input Array B Output Array C 
void Merge(int A[], int B[], int C[])
{
  int I = 0, J = 0, L = 0;
  while(I < A.length && J < B.length)
    if(A[I] < B[J])
      C[L++] = A[I++];
    else
      C[L++] = B[J++];
  while(I < A.length)
    C[L++] = A[I++];
  while(J < B.length)
    C[L++] = B[J++];
}
