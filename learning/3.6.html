Booleans

Booleans Data Type

The boolean data type is used for logical values. The boolean data type can only have two possible values, true or false. It can be used in order in logical statements to determine whether they are true or false.

Below you can see how we might initialise different boolean values:


boolean b1 = true; //initialising a variable b1 which is assigned a value of true
boolean b2 = false; //initialising a variable b1 which is assigned a value of false

Now you might be wondering how we can use booleans in logical statements and why they are so important. The following examples below should make this clearer.

Below we firstly initialise booleans and give them the values "true" or "false". In the bit after we are using the "==" operator which checks whether the two variables are equal or not, if they are equal it returns "true" if not, it returns "false"

public class Booleans {
    public static void main(String[] args) {

        boolean w = true;
        boolean x = false;
        boolean y = false;
        boolean z = true;

        System.out.println(w == z);
        System.out.println(x == y);
        System.out.println(w == x);
    }

}

This is what your terminal should output:

true
true
false

Below we show you another way you can compare values. This is not really to do with booleans but is to do with operators and checking things that can be true or false. For example, below we have initialised some numbers and are checking whether one is bigger, smaller or equal to the other. If the condition for the operator is true, it returns "true" else it returns "false" is the condition is not met.

public class Compare {
    public static void main(String[] args) {

       int a = 5;
       int b = 10;
       int c = 100;
       int d = 100;

        System.out.println(a > b);
        System.out.println(a < c);
        System.out.println(c == d);
    }

}

This is what your terminal should output:

false
true
true
