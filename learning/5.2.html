Loops

Loops in programming languages are used for repeating the execution of the same statements again and again. In Java, there are three loops, for-loop, while loop, do-while loop.


for loop
while loop
 
Syntax:

for (initialisation; condition; updation)
    Statement;
 
Syntax:

while (condition)
    Statement;
Example: 
for(int 1; I <= 5; I++)
    System.out.print(I + “:”);

do-while loop
 
Syntax:
do
{
    Statement;
}
while(condition);
 
It is an Exit Controlled Loop, i.e. condition is at the  end. This 
loop is ideal for a situation, when you require to execute the loop 
statement(s) at least once and the number of iterations (repetition 
of statement(s)) is not known at the time of starting of the loop. 
Curly braces { } are essentially required even for a single statementof the loop.
 
Example 1:

int I = 1;
do
{
    System.out.print(I++ + “:”);
}
while(I<=5);

Example: Write program for reversing an integer

Solution:

int N, TN, Rev = 0;
System.out.print("N: ");
N = s.nextInt();
TN = N;
while(TN > 0)
{
  Rev = Rev * 10 + TN % 10;
  TN /= 10;
}
System.out.println("Reversed " + Rev);

Reversed 4321

Example: Write a program for disbursing the fund as per various claims on first come first serve basis till the fund is more than 0.

Solution:

int claimNo = 1;
float Fund, Claim;
System.out.print("Fund: ");
Fund = s.nextFloat();
do
{
  System.out.println("Available Fund: " + Fund);
  System.out.println("Claim No: " + claimNo++);

Nested Loops

Using a loop inside the other loop is known as nesting of loops or nested loops.

Example:

int N;
System.out.print("Enter N: ");
N = s.nextInt();
for(int i = 0; i < N; i++)
{
  for(int j = 1; j <= N; j++)
    System.out.print(j);
  System.out.println();
}

Break and Continue

Break: The break statement allows to user to break the loop and directly exit it to continue executing the code after the loop.

Example

int x = 10;
while(true)
{
    // Break the loop when value of x is 0
    if(x == 0)
        break;
    x--;
}

Continue: The continue statement allows the user to break one iteration of the loop, and jump to the next iteration of the loop.

Example

int x = 10;
while(true)
{
    // Skip the iteration when value of x is 5
    if(x == 5)
        continue;
    x--;
}
