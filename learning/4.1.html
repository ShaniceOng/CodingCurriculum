Writing Your First Program!

With any programming language, every beginner starts their journey with a hello world program. This is displayed below 



    public class HelloWorld
    {
      public static void main(String[] args)
      {
        System.out.println("Hello World");
      }
    }

How the program works

The standard Java program will have an anatomy similar to the hello world program. Each java program has a class. This will be covered in more advanced lectures, but for this case, a class is template that allows a program to perform a certain behaviour. The basic convention of naming a class uses an upper case letters to name every new word.The class name also has to start with the upper case letter. Each class has methods within it. In this case, methods allow the class to carry out a certain behaviour mentioned above.

Depening on your program, there can be many classes, and each class will have many methods that allow it to perform many different things. However, every class requires a main method. All the other methods are called through the main method, therefore a program needs a main method to work.

The main method is titled "public static void main(Strings[] args)." The key word "public" means that the method is accessible from outside the class; it can be used by other classes. For now, the keyword "static" is not important and will be referred to in other lectures. The key word void means that the method does not have a return type; this means that once it is called, it does not output any of the specified data types mentioned before. The keywords in the brackets mean that the method takes a string as a parameter; this means that for the method to work, it needs a certain set of values to work, and in this case, these values are strings.

Looking within the main method, The statement used to print the words "Hello World" is called a function. The System.out.println is a function that allows a user to display some information on the screen. Usually, this information is placed within brackets and if it is a set of words, it is surrounded by double quotes " " because the system will not understand words such as hello world. Additionally, the function ends in a semi-colon(;). This is necessary because it tells the system when to stop. The System.out.println("Hello World") is an example of a statement and every statement in java ends with a semi colon. A statement is the Java code you write to tell the computer what to do. It is like a sentence in a language.
