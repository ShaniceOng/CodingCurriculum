Characters

Data Types

When you make a variable in a programming language, you are reserving memory locations in order to store values. So everytime you make a variable some space is resevered in memory.

Depending on the type of data type of a variable, the operating system can allocate memory and decide what should be stored in the memory.

Depending on the data type you can store integers, characters, floats etc in these variables.

Before we use any data types, they have to be initialised or declared. We do this by putting the data type followed by an equals sign and then what value the data type takes.

Character Data Type

A character is a single letter of the alphabet, number, space, punctuation mark or symbol. It is usually enclosed within apostrophes to show that it is being used as a character.

Here is how you could initialise a character data type:


char ch = 'a'; //initialising a letter
char ch = 'b';
char ch = 'c';
char ch = ' '; //initialising a space
char ch = '1'; //initialising a single digit
char ch = '!'; //initialising a character

Escape characters

A character that has a backslash ( which looks like this \ ) before it has a special function when compiled. For example \n is known as the new line character and creates a new line.

Escape Seqence Character What it does
\t Inserts a tab
\b Inserts a backspace
\n Inserts a new line
\r Inserts a carriage return
\f Inserts a form feed
\' Inserts single quote
\" Inserts inserts double quote
\\ Inserts backslash
