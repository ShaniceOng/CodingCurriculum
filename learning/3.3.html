Float

Float Data Type

A float is used to represent a number with more accuracy. It often has a maximum precision of 7 decimal places so is useful for when you need to work with fractional parts of numbers. When initialising a float variable we have to put an 'f' after the float number to show it is a float.

Here is how you could initialise a float data type:


float a = 10.2f; //initialising a floating point number, notice the f after the 10.2

Here is a sample code of how floats can be used in Java. Copy and paste this into your IDE and play around with the numbers and perhaps initialise and add more floats and see what you can do with them.

Now if we go back to our older example where we used the equation for force. We were using "int" so we could not get numbers with decimals in them. Normally in equations like this we need to work to more decimal places, so having a float will allow us to do this. Take a look below at how it is done.


public class Floats {
    public static void main(String[] args) {

        int mass = 3.2f;
        int acceleration = 17f;
        int force = mass * acceleration;

        System.out.println("The force is: " + force + " N");
    }
}

This is what your terminal should output:

The force is: 54.4 N

Make sure you try change the numbers around and see how the output differs.
