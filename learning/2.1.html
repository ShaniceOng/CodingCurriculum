Comments

Sometimes when you are writing code, it can all get a little cluttered and you might not be sure what bits do what. Comments are lines of writing that are missed out by the compiler, so they are useful way for you explain what your code does.

There are two kinds of comments- single line and multiline comments. Single line comments only add comments for one line whereas multiline comments let you comment out as many lines as you need. Make sure not to go overboard with adding comments to your code or you might end up making everything even more confusing.

This is an example of how you can use single line comments. Everytime you type in // the whole line will be commented out and not executed. Anything that does not have it will be run.


//This is an example of a single line comment

//The code on this line will not be executed

//The code below do not have the '//' tag so it will be executed

package com.test;

public class Main {

    public static void main(String[] args) {
      System.out.println("Hello World");


    }
}

Sometimes you might need to write in longer comments and it is not ideal to have to type in // for every line. Instead you can use /* */ and type in whatever you want sandwiched in between, like in the example below:



//If you are going to type in a long comment
//it might be annoying to have to 
//type out the double slash every
//single
//time
//instead you can try something like this:

/*
This is an example of a multiline comment
You can write out everything you want as a 
comment within these tags /* text */
It means you don't have to keep writing out
a double slash after every line
*/

Below is an example of how you might use single line and multiline comments in your code to make it easier for other people to understand or make it easier for you to see what part of your prgoram does what


/*
This program is mean to show use how to access method
and instance variables
*/



public class Dog {
    int dogAge;

    //This part will set the name of the dog
    public Dog (String name){
        System.out.println("Name: " + name);
    }

    //This part will set the age of the dog
    public void setAge (int age) {
        dogAge =age;
    }

    //This part will get the age of the dog
    public int getAge (){
        System.out.println("Age is: " + dogAge);
        return dogAge;
    }

    //This part will print the details of the dog
    public static void main (String []args){
        Dog dog1 = new Dog ("James");
        dog1.setAge(10);
        dog1.getAge();
        System.out.println(dog1.dogAge);
    }

}
