How to set up before you get started!

Step One...

Open up IntelliJ IDE, we have never created a project before, so we will have to select "Create New Project" from the welcome screen. Now we will have to select "Java" from the left-hand pane.


Step Two...

Now it may say "none" next to the Project SDK dropdown, that means we need to select the JDK for IntelliJ to use. Click "New" and select "JDK" from the dropdown.


Step Three...

In the new dialog box that opens, you need to find where the JDK is installed. For windows this is typically "C:/Program Files/Java/jdk...


Step Four...

For now do not select anything under "Aditional Libraries and Frameworks" because for now we are going to make a simple application. Just click "next" at the botom.


Step Five...

We are going to get the following dialogue box now, don't select any of the options and click "next" again. The next page should give you the option to put in a project name, in the box type out "HelloWorld" as this is what our first application will be called. Later when you are used to Java, you can call it whatever you want.



Step Six...

Once IntelliJ has made your product, it should look like this. Under the project, there should be a folder ".idea" and within that a folder called "src".


Right-click on the "src" folder and create a new Java Class. In the dialogue box, type in "com.example.helloworld.HelloWorld" for the name.


If you can see the class opened up in the editor as below, you are ready to go!
