A bit more about us!

What is this website for?

The purpose of this website is to give you an insight into Java. We assume none or little prior programming experience, so you can pick up our lessons no matter what stage you are at and we try to explain everything as clearly as we can while using examples so that at every stage you will feel confident and excited to learn more. We also follow a STEM theme so that we can link what you learn in Java with real world applications and show you why Java is so useful.

Theme

We follow a general STEM theme for our project. We do not simply stick with one subject but explore a variety of different areas such as Physics, Chemistry, Biology, Arts and Engineering. The theme will be obvious from the exercise or the lesson, however do not be worried if you have not studied the subject matter before since we will make sure to explain everything right from the beginning so that you are able to apply your newly learnt Java skills to that field.

Scope

The project is broken down into:

19 Lessons: You can learn the basic buidling blocks you need to start getting a grasp of Java and it's concepts, make sure to go through all of these.

Exercises: Here you can test what you have learnt, we provide easy, medium and difficult challenges so that you can slowly work up your skills and confidence and then perhaps try stretch yourself a little toward the end. We will provide answers for all the core questions.

Final projects: Having tried out the exercises, it is then time to try something exciting! Projects are a great way to take your newly acquired skills and use them to produce something really interesting. In our projects section, we start off with a warm-up project, a simple program to calculate the greatest common divisor of as many numbers as you want in seconds! After that, have a look at our Tower of Hanoi Project, challenge yourself and try to solve this famous puzzle with... Java! Then, we move on to our main projects, make your own converter application, make a simple calculator application and learn about collisions in Swing; these are skills that will start you off on building your own Java game!

Learning Outcomes

Understand the basic syntax of Java

Be comfortable using an IDE

Use simple programming techniques to build simple programmes

Use your knowledge to build something substantial

What you need

A laptop/computer

An IDE for Java installed, preferably the one we use in tutorials

Latest version of Java installed
