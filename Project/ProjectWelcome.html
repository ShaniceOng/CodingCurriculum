Welcome to the project

The project is broken down into a series of workshops that will enable you to produce a game using all the different building blocks we will give you.

There is no fixed game at the end, so what you create is totally down to you. We teach you how to make the backgrounds, the characters, the interactions and how to put in the sounds and then it is down to your imagination!

We will be using Swing in order to produce a game and will use a lot of key concepts we have covered in previous learning exercises to do so. Before you get started we are going to explain what Swing is and how you can get it up and running.

So what is Swing? Swing is a set of program components that will allow you to create a graphical user interface (GUI)- these components are used with Java Foundation Classes (JFC). Swing is light weight meaning that the swing components are independent of native operating system's API since Swing API controls are mainly rendered using JAVA code instead of underlying OS calls. Swing has rich controls such as colorpickers, tabbedPane etc. It is highly customisable and it is also very easy to do so.
