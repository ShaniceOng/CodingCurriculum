Basic Converting App

Buidling up our program a little more

Using NetBeans IDE, we are now going to build up what we can do with Swing. Simply printing strings won't get us very far, we need to do something more interesting. So let's make an example program where you can input a number and convert it to something else you may need!

First let's make a new project in NetBeans, open up NetBeans and in the top-left corner click "file" then select "New Project..." Then select "general" and then select "Java Application" in the left hand pane. You can now choose the project name, set this as "ConvertingApp" and deselect the "Create Main Class" option. Finally, click "Finish"




Now first we need an area within which we will work, the physical area where our components will go. To do this, we add a JFrame Form, go to the ConverterApp name and then right-click and then choose New -> JFrame Form. This is the main frame of your application as previously stated. In the Class Name section write in "ConverterAppGUI" and the package name as "MyApp" (this can actually be whatever you want it to be, but this is our own convention for our lessons). Now click "Finish". You will notice a frame has been drawn onscreen, this is where we will drag and drop our components.





NetBeans makes designing a GUI much easier via it's drag and drop mechanism. The GUI is shown in the "Design" tab and the auto-generated source code is shown in the "Source" tab. For now we are going to work using the design tab.

First we are going to set the application title by setting JFrame to "ConverterApp". To do this click on JFrame in the inspector and then in the properties tab change the title to "ConverterApp".



Next, add a JTextField from the palette into the JFrame. The GUI builder will give you guidelines as to how the field will fit into the frame. Then add a JLabel to the right of the JTextField, again using the guidelines to decide on spacing. Next, add a JButton below the JTextField and then add another JLabel to the right of the JButton.



Right now our GUI does not look very interesting so we will adjust sizes, IDs and the text. Right click on the components and select "Edit Text" in order to change what the text says. Change the button text to "Convert", change the text field to nothing, i.e. delete the current text. For the top label change it to grams and then for the bottom label change it to kilograms.



We now need to make the size of the button the same as the text field, so shift-click on both the button and the text field, then right-click and select "Same size -> same width". This makes the width of both the components the same. At this stage you can remove any extra space around the app by resizing you frame, to do this just go to the corners or edges and resize as needed. Well done, you now have a GUI! Except it doesn't actually do anything right now, so the next part is adding the program logic.

When a user will interact with a Swing GUI component, in our case clicking the button, the component will generate a new object, an event object that broadcasts to objects that have registered themselves as listeners to that event. Right-click on the convert button, click "Events -> Action -> ActionPerformed". This will have now generated event-handling code, with an empty body for your to add in your own code.

Congratulations, you have now built your first application with Java that uses a GUI! To run the project, just click "Run -> Run Project".
