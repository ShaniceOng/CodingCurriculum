private ArrayList<Treasure> treasureList;

As every object of this class will have its own list of treasures and will make a list of the quantity of each treasure, to be used in its methods to calculate GCD, we write these as instance variables in the class. GCD (ArrayList<Treasure> treasureList)
{
  this.treasureList = treasureList;
  listLength = treasureList.size();
  quantities = new ArrayList<Integer>();

This is the constructor for this class. Whenever we want to create an instance of this class, we will use this constructor. Our constructor also creates a new ArrayList for the "quantities" ArrayList for every instance of the class.


public ArrayList<Treasure> getTreasureList()
{
  return treasureList;
}

This is our getter method for the private treasureList attribute. Recall that, to implement encapsulation, we have to make our attributes private and only accessible through public getter and setter methods.


private void addQuantities()
{
  for (Treasure treasure : this.getTreasureList())
  {
      quantities.add(treasure.getQuantity());
  }
}

As the quantities array list that we have created in the constructor is initially empty, we have to populate it using this addQuantities method, which loops through the treasure list to collect all the quantity values and add them to the quantities list.

The add method adds the element to the next index of the array list. We could have also used add with an index, quantities.add(index, value), allowing us to add a value to any specific location in the ArrayList. However, ArrayLists can only be used with class types, not primitive types. Still confused about ArrayLists? Check out this tutorial on ArrayLists!


public int getGCD()
{
  this.addQuantities();
  int firstQuantity = quantities.get(0);
  int remainder;
  for (int i=1; i<listLength; i++)
  {
    int secondQuantity = quantities.get(i);
    if (secondQuantity>firstQuantity)
    {
      int quantityTemp = firstQuantity;
      firstQuantity = secondQuantity;
      secondQuantity = quantityTemp;
    }
    remainder = firstQuantity%secondQuantity;
    while (remainder>0)
    {
      firstQuantity = secondQuantity;
      secondQuantity = remainder;
      remainder = firstQuantity%secondQuantity;
    }
    firstQuantity = secondQuantity;
  }
  return firstQuantity;
}

This is an important method in the GCD class because it uses the other methods to calculate the GCD of the quantity values (the main goal of this project). Here, we are implementing Euclid's Algorithm, the algorithm that we explained in the video in the previous section.

private String getString(String string)
{
  System.out.print(string);
  Scanner inString = new Scanner(System.in);
  return inString.nextLine();
}

private int getInt(String string)
{
  System.out.print(string);
  try {
    Scanner inInt = new Scanner(System.in);
    return inInt.nextInt();
  }catch(InputMismatchException e) {
    System.err.println("Please enter an integer");
    return -1;
  }
}

Both of these methods deal with user input. getString takes and returns a string input while getInt takes and returns an integer input. The string argument is passed into both methods is the question that is printed out to prompt the user for an input. The "try and catch" block prevents the program from terminating when the input received is not an integer. It catches that error and returns a "-1" instead of terminating. This means we are able to write code to give the reader another chance to type an integer in.

Exception Handling is a very important part of Java programming, especially for longer and more complex programs, as they either prevent the program from terminating or allow the program to terminate gracefully and, more importantly, tell us where the error is so we can fx it.


import java.util.InputMismatchException;

As we are using the InputMismatchException type to catch the error object and the Scanner to take user input, we should insert these "import" commands at the top of the file, uunderneath the package statement.

Input Mismatch Exception is thrown by a scanner when the scanned item does not match the pattern for the expected type.

private int printOptions()
{
  System.out.println(ADD_TREASURE + ". Add a type of Treasure");
  System.out.println(GET_NUM_TREASURE + ". Get number of each treasure per person");
  System.out.println(GET_NUM_PEOPLE + ". Get maximum number of people");
  System.out.println(QUIT + ". Quit");
  return getInt(">> ");
}

This method prints the options menu to the console for the user to pick their option by submitting the corresponding numbers.


private void getOptions()
{
  int option = ADD_TREASURE;
  while (option<QUIT && option>=ADD_TREASURE) {
    option = printOptions();
    switch (option) {
      case ADD_TREASURE:
        //allow user to add treasure here
        break;
      case GET_NUM_TREASURE:
        //tell the user the amount of treasure each person can get
        break;
      case GET_NUM_PEOPLE:
        //tell the user the maximum number of people Dr Java can distribute the trasure to
        break;
      case QUIT:
        break;
      default:
        System.out.println("Invalid input. Please try again.");
        option = 1; //to allow the user to enter an option again (repeats the loop)
        break;
    }
  }
}

The getOptions method calls printOptions, takes the value returned by printOptions (the user input) and uses this value in a switch statement to decide which methods to call next. Right now the blocks within the first three statements are just filled with comments. Let's write some methods that we can use when users choose these options.

private int calculateGCD()
{
  GCD gcd = new GCD(treasuresList);
  return gcd.getGCD();
}

The calculateGCD method simply creates a new gcd object, passing the current list of treasure into its constructor, and calls its method to calculate the GCD of the quantities of all the treasure. This method is used in several other methods.


private void getTreasureCollection()
{
  int divisor = calculateGCD();
  System.out.println("Treasure combination for each person:");
  for (Treasure currentTreasure : treasuresList)
  {
    int treasureQuantity = currentTreasure.getQuantity();
    int perPerson = treasureQuantity/divisor;
    System.out.println(currentTreasure.getType() + " = " + perPerson);
  }
}

This method is used for the second option. It finds the GCD by calling calculateGCD then loops through the current list of treasure to get the amount of each treasure that each person should get. It also prints the result to the console.
