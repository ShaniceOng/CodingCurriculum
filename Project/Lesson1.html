Basic Controls

UI elements

UI elements refers to the visual elements that the user will see onscreen and interact with. There are a huge range of common elements to choose from which you will encoutner during this project.

Layouts

Layouts define how the UI elements will be organised on the screen and help to provide the final look and feel of the GUI.

Behaviour

This desribes what happens when the user interacts with the UI elements- such as a button click.

Component -> Container -> JContainer

A component is the abstract base class for the non menu UI controls of Swing. Compenet is the object with a graphical representation. A container is then a component that can contain other swing components. A JComponent is a base class for all swing UI components.

How to compile and run Swing Programs

We are going to be executing Swing programs from the command line in the next project section- so we will teach you how to do that here. However, before we get to that make sure that you have installed the latest version of the Java SE platform. Whenever we want to run a program, we will first creat the program that will utilise the swing components, compile the program and then run it.

Firstly once you have written a java program, save it as a .java file.

Then open up your command line terminal and go to the directory where it is saved. You can use cd + the directory name to navigate up to the place you have saved the java file. Now type in "javac filename.java" in order to compile the code. If there are no errors, it means your Java code has compiled successfully.

javac filename.java

Now to run the Java code, you must type in "java filename". This will execute the program and a little window should pop up where your code has run- the GUI. 

java filename

In the exercises after the "Hello World" part, we will be using the NetBeans IDE, this is much better than using a simple text editor, much the same way we had used the IntelliJ IDE earlier. So make sure you have installed NetBeans before starting.
