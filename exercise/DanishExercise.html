Answer To Question 1 

  public static void InsertionSort( int [ ] array)
    {
          int i;                   // the number of items sorted so far
          int j;
          int key;                // the item to be inserted
            

          for (j = 1; j < array.length; j++)    // Start with 1 (not 0)
            {
                key = array[ j ];
                for(i = j - 1; (i >= 0) && (array[ i ] < key); i--)   // Smaller values are moving up
                  {
                      array[ i+1 ] = array[ i ];
                  }
                 array[ i+1 ] = key;    // Put the key in its proper location
             }
        }

Answer To Question 3

The Code For The Program 

 


public static int maxValue(int[] array) 
{
  int max = array[0];
  for (int i = 0; i < array.length; i++) 
  {
    if (array[i] > max) {
      max = array[i]; 
    }
  }
  return max;
}

How the program works

The program first declares the method maxValue which takes an array as a parameter and returns an integer. The program begins by instantiating an integer max with the first element in the array. In the for loop which terminates at the end of array, each element in the array is compared to max. If there is a value greater than max, then max is assigned that value. By the time the array has been traversed at the end, we would have ensured that the value of max is the largest value in the array, which would be later returned.

      for (int i = 1; i <= N; i++)

      if(i % num == 0)
      {
    count = count + 1;
      }
   }
   if (count == 2)
   {
      //Appended the Prime number to the String

   }  
      } 
      System.out.println("Prime numbers from 1 to N are :");

   }
}

How the program works


The program begins by importing the Scanner class in order to read input from the user. After the declaration of class and the main method declaration the program, the object 'input' of class Scanner was instantiated. After that, a String 'primeNumbers' was declared as an empty string. The user was then prompted of an input by printing "Enter the value of N:" on the screen, after which the input was read by input.nextInt() and then assigned to a newly declared and immediately instantiated, int N. After that, a loop which began at 1 and terminated at N was began to iterate through every number between 1 and N. In order to determine weather the number a number X prime or compisite, we have to check how many numbers from 1 to X divide it. In order to determine the quantity, we need to declare an external counter variable count, and then created another 'num' loop (nested in previous 'i' loop) to check the divisbility of numbers from 1 to X. If the modulo is zero (i.e. the number is divisible), the counter is incremented by one. Therefore, for a prime number, the counter should always be 2 (as it has only two divisors: one and itself). A string is instantiated which concatenates the primeNumbers string, the number, if the count = 2 (i.e. the number is prime). The loop is terminated here, and the numbers are printed as a collective string separated by spaces after being outputed under the output line 'Prime numbers from 1 to N are :'. For example, if the user had inputted 'N' as 10, the output would be: Prime numbers from 1 to N are : 2 3 5 7
