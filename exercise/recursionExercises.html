Recursion Questions!

  1. Today’s problem will be focusing on mathematics. I’m sure you’ve done basic addition. Problems like 3 + 4, 20 + 30, or even 100+1490 should be pretty easy right now. Today we’re going to program a special pattern of numbers using addition called the Fibonacci Series. The way this sequence works is that the next number is the sum of the previous two numbers. Still confused? Take a look at the Wikipedia page: https://en.wikipedia.org/wiki/Fibonacci_number. Your task is to write a program that finds the nth number of the Fibonacci series using recursion.

  2. I’m sure you’ve all realized how important mathematics is, so we’re going to do another math’s related question: This time will be tackling a very important division problem. Let’s start with some basic terms: a number is a divisor of another if it divides the other number evenly (without a remainder). The greatest common divisor(GCD) of a pair of numbers is the largest number that divides both those numbers evenly. For example, the GCD of (40,50) is 10. Write a program that uses recursion to calculate the greatest common divisor. A look at this page may help: https://www.khanacademy.org/computing/computer-science/cryptography/modarithmetic/a/the-euclidean-algorithm

Answer To Question 1

The Code For The Program 



public static int Fibonacci(int n)
{
  if(n == 0)
    return 0;
  else if(n == 1)
    return 1;
  else
  return Fibonacci(n - 1) + Fibonacci(n - 2);
}

How the program works


As you can see, the code above uses recursion. If you followed up on the Wikipedia page, or understood how the Fibonacci sequence works, it starts like 0,1,1,2 and etc. As you can see above, the base case is anything less than 1 because the sequence will not progress if a number larger than 1 isn't input. If a number such as 3 is input, it will be Fibonacci(2) + Fibonacci(1). Recursion will call the function again and it will produce 2.

Answer To Question 2

The Code For The Program 

 

public static int gcd(int num1, int num2)
{
  if(num2 == 0)
  { 
    return num1;
  }

  else
  { 
    return gcd(num2, num1%num2);
  }
}

How the program works


If you've understood the concept of GCD, or followed up with link in the question, you'll notice that the program uses recursion to divide the first number and the second number. It then calls the method again using the second number, and the remainder of the division of the first and second number. It continues this process until the remainder is 0.


Let's take an example of 10 and 12. Since the second number isn't 0, the method will be called with 12 and the remainder of when 10 is divided by 12. So it is called with 12 and 2. When 12 is divided by 2 the the remainder is 0. Therefore, 2 will be returned by the method.
