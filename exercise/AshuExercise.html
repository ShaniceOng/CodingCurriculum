Answer To Question 1


The print statement is used to print to the console, however it will not start on a new line, but continue on the pre-existing line. The println statement will specifically start on a new line every time the command is executed. The printf statement, like its name suggests, is used to print to the console, however it provides more focus on what type of data is printed. For example, it is used to specify what type of data is printed e.g. strings, integers, doubles and can also provide formatting options to certain types of data like number of decimal places



Answer To Question 2

The Code For The Program 

 

import java.util.Scanner;

public class exercise1 {

  public static void main(String[] args) 
  {
    String name; //declaring a variable to hold your name
    Scanner in = new Scanner(System.in); //opening a new input stream
    System.out.println("please enter your name");
    name = in.nextLine(); //asking for input fromt the user, this is where their name is stored
    in.close(); //closing the input stream
    
    System.out.println("Hello " + name + ", my name is gordon, it is nice to meet you");

How the program works


Notice the first line of the program. This import statement is used to refer to external functions that are not part of the normal java library. This means that the input function we want to use is not part of the normal functions in the program, so we have to get the function from somewhere else.


The rest of the program is written in the main method. The users name is stored in a variable called string. Here the scanner class is what is actually used to get input from the user. You are created a variable called in, which is used to initially store input from the user. The users input is stored in the name variable. The in.nextLine() function is used to show that a string is being entered by the user. It is stored here, before it is passed onto the the name function. Finally, the in.close() is used to ensure that no more data is being input by the user. At the end, the program prints the sentence to the console.

import java.util.Scanner;
    double distance = 0; 
    double time = 0; 
    double avgSpeed = 0;
    double[] speed = new double[5];
    Scanner in = new Scanner(System.in);
    
    System.out.println("enter 5 different average speeds");
    for(int i = 0; i < 5; i++)
    {
      speed[i] = in.nextDouble();
    }
    
    for(int i = 0; i < speed.length; i++)
    {
      avgSpeed += speed[i];
    }
    
    avgSpeed = avgSpeed / speed.length;
    
    System.out.println("enter distance taken to fly to destination");
    distance = in.nextDouble();
    
    time = distance / avgSpeed;
    
    System.out.println("the time taken to get to your holiday is:" + time);

How the program works


Yet again, this has similar elements to the previour programs because the scanner class is required. The first few variables are declared normally, however any array is created to store the averages speeds. After the in variable of the scanner class is created, a for loop is used to add input values to the array. The for loop after that is used to add up all the speeds, after which the speed is divided by the number of elements in the array. The next lines just involve calculations and print statements.
