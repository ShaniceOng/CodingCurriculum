  3. To check if a number entered by the user is a palindrome or not.

    Solution and Output

    int N, TN, Rev = 0;
System.out.print("N: ");
N = s.nextInt();
TN = N;
while(TN > 0)
{
  Rev = Rev * 10 + TN % 10;
  TN /= 10;
}
if(N == Rev)
  System.out.println("Palindrome");
else
  System.out.println("Not Palindrome");


    
N: 1223221
Palindrome
